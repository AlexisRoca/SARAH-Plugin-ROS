Le plugin ROS permet d'envoyer des messages au server rosbridge_suite de ROS

Différents paramètres sont à régler avant d'utiliser le plugin, il faut les rentrer dans le fichierROS.prop :

{
  "modules": { 
    "ros": {
      "description": "Envoyer des messages sur un topic ros",
      "version": "1.0",
      "ip": "192.168.1.52",
      "messageType" : "std_msgs/String",
      "topic" : "/base_controller"
    }
  }
}
      
Il faut d'abord entrer l'adresse de son robot sous ROS dans la partie ip

      
ensuite il faut entrer le type de message que l'on souhaite envoyer dans la partie messageType
      
      
et enfin le nom du topic dans lequel on veut publier les messages dans la partie topic Attention, ne pas oublier le '\' avant le nom du topic

      

      
Explication du plugin:

      
Le plugin permet d'envoyer d'envoyer des messages sur un topic que l'on aura défini, ces messages pourront être traité dans un listener placé dans le robot (pour plus d'information, il faut aller voir les liens en bas de page)
